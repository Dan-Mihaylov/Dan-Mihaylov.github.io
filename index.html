Studying Software Development With Python At Softuni

SOFTWARE DEVELOPEMENT ROADMAP

Python Basics

Initial programming skills necessary. Writing entry-level programming code working with a developement environment (IDE), using variables and data, operators and extenssions, working with the console (reading input and printing results), using conditional statements (if, elif, else) and loops (for-loop, while-loop).

Python Fundamentals

Programming fundamentals examines the fundamental concepts in programming, which are the basis for in depth understanding of programming. Building on top of the knowledge already gained in the programming basics and learning new and more complex programming concepts and structures such as:

Lists, Dictionaries, Objects and Classes.

Familiarization with the additional technologies needed by every software developer, such as HTML and CSS, HTTP, bitwise operations and more. A glimpse at the fundamentals of Web Developement, as well as the most common technologies and tools for the purpose.

Python Advanced

The Python Advanced course will build on the Python language skills by exploring more complex concepts typical for the language. Students lear about creating linear data structures, develop problem solving skills by solving algorithmic problems, work with stacks and queues, tuples and sets (Stacks & Queues, Tuples & Sets), matrices (Multidimentional Lists), as well as files and directories.

Familiarization with functional programming paradigm. The course has a deeper look at rtecursive functions, functions with multiple arguments (*args) and multiple keyword aguments (**kwargs).

Python OOP

Python OOP is teaching students wht principles of object-oriented programming (OOP), working with classes and objects, using object oriented modeling, and building class hierarchies. Basic OOP principles such as Abstraction, Polymorphism, Encapsulation and Inheritance will be studied. They will enter the most frequently used design patterns (creational, structural and behavioral design patterns). Introduction to SOLID principles and learning about iteratiors, generators and how to use them. Attention will be given to component testing (writing unit tests) and the concept of Test Driven Developement (TDD).
