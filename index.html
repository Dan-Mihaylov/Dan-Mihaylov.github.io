SOFTWARE DEVELOPEMENT ROADMAP

Python Basics

Initial programming skills necessary. Writing entry-level programming code working with a developement environment (IDE), using variables and data, operators and extenssions, working with the console (reading input and printing results), using conditional statements (if, elif, else) and loops (for-loop, while-loop).

Python Fundamentals

Programming fundamentals examines the fundamental concepts in programming, which are the basis for in depth understanding of programming. Building on top of the knowledge already gained in the programming basics and learning new and more complex programming concepts and structures such as:

Lists, Dictionaries, Objects and Classes.

Familiarization with the additional technologies needed by every software developer, such as HTML and CSS, HTTP, bitwise operations and more. A glimpse at the fundamentals of Web Developement, as well as the most common technologies and tools for the purpose.

Python Advanced

The Python Advanced course will build on the Python language skills by exploring more complex concepts typical for the language. Students lear about creating linear data structures, develop problem solving skills by solving algorithmic problems, work with stacks and queues, tuples and sets (Stacks & Queues, Tuples & Sets), matrices (Multidimentional Lists), as well as files and directories.

Familiarization with functional programming paradigm. The course has a deeper look at rtecursive functions, functions with multiple arguments (*args) and multiple keyword aguments (**kwargs).

Python OOP

Python OOP is teaching students wht principles of object-oriented programming (OOP), working with classes and objects, using object oriented modeling, and building class hierarchies. Basic OOP principles such as Abstraction, Polymorphism, Encapsulation and Inheritance will be studied. They will enter the most frequently used design patterns (creational, structural and behavioral design patterns). Introduction to SOLID principles and learning about iteratiors, generators and how to use them. Attention will be given to component testing (writing unit tests) and the concept of Test Driven Developement (TDD).

PostgreSQL

Databases are the standard way of using dynamic information with modern technology, making them an indispensable part of almost every application. This course will introduce students to the fundamentals of database management system (DBMS) used in the developement of modern information systems - PostgreSQL. The course gives serious attention to relational model, ER diagrams (tables and relational relationships) and working with the SQL language (data extraction, selection, projection, joins, aggregation, grouping, modification, deletion and insertion). Will gain fundamental knowledge of working with functions, procedures, transactions, and triggers.

This course will lay the foundations of working with databases, making it easier to use them in real-world projects and understanding the concepts such as converting a database into an object-oriented-model (ORM).

Python ORM

Python ORD covers the basics of mapping between a relational database and an object-oriented model (ORM) with a focus on the Django framework. It introduces the students to the ORM and its key concepts and then moves on to working with models in Django, including defining models, fields, and methods. Advanced topics such as migrations are covered as well as the Django admin interface, query operations, model relationships and inheritance. Another popular ORM framework is shown as well, SQLAlchemy, which offers an alternative approach.

After the training, participants will acquire the basic skill set to build robust, scalable and efficient database-driven web applications using Python ORM frameworks.

Python Web Developmennt

In the "Python Web" module, students delve into the fundamentals of web-based technologies, with a particular focus on Django, the preeminent framework in the Python ecosystem. Additionally, the curriculum encompasses the acquisition of essential skills for managing dynamic data through database interactions. The coursework encompasses a progressive journey, commencing with the construction of basic web applications necessitating familiarity with the HTTP protocol and advancing towards more intricate concepts.

Throughout the program, students are tasked with designing web applications that address real-world challenges, thereby fostering a profound comprehension of the technology's capabilities. The module culminates with students developing and safeguarding their own web applications, marking a significant milestone in their pursuit of a career as software developers.

JS Front-End Developement

The "JS Front-End" module is designed to impart fundamental proficiency in HTML, CSS, and JavaScript. Participants will acquire a foundational understanding of web development, web browsers, and the utilization of development tools for HTML/CSS/DOM. This course fosters practical expertise in employing HTML, the language for describing web content, and CSS, the language for styling web content. Students will gain the competence to effectively structure the Front-End component of web applications and apply styling techniques in accordance with provided design specifications.

In the latter part of the module, students will delve into the rudiments of the JavaScript language, encompassing essential syntax principles, array and object manipulation, class construction, and function authorship. The curriculum also fosters algorithmic thinking and equips learners with the capacity to navigate the Document Object Model (DOM), effect modifications to it, and respond to user-generated "events." The module encompasses both functional and object-oriented programming paradigms in JavaScript. Additionally, students will explore concepts related to HTTP and REST data transfer.
